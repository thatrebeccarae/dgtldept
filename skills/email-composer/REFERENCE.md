# Email Composer — Reference

## Email Templates

### Cold Outreach

```
Subject: [Specific value proposition for their situation]

Hi [First Name],

[One sentence showing you researched them — reference something specific about their company, role, or recent work.]

[One sentence about the problem you solve that is relevant to them.] [One sentence about the result, ideally with a number.]

[Specific, low-friction CTA: "Would a 15-minute call next week make sense?" or "Would it be helpful if I sent over [specific deliverable]?"]

Best,
[Your Name]
```

**Rules:**
- Under 100 words
- No attachments on first touch
- Never lead with "My name is..." or "I'm reaching out because..."
- Reference something specific about them (not generic flattery)

### Warm Follow-Up (After Introduction)

```
Subject: [Mutual contact name] suggested we connect

Hi [First Name],

[Mutual contact] mentioned you're [working on / dealing with / interested in] [specific thing]. [One sentence connecting your expertise to their need.]

I [helped / worked with] [similar company/situation] and [specific result].

Would [specific, easy next step] be useful? Happy to [offer something of value first].

Best,
[Your Name]
```

### Deliverable Handoff

```
Subject: [Deliverable name] — ready for review

Hi [First Name],

Attached is the [deliverable name] covering [scope summary].

Key findings:
- [Finding 1 — the most important insight]
- [Finding 2]
- [Finding 3]

[One sentence about recommended next steps or what you need from them.]

Let me know if you have questions — happy to walk through it on a call.

Best,
[Your Name]
```

```

### Scope Change Discussion

```
Subject: [Project name] — Scope adjustment for [reason]

Hi [First Name],

During [work/analysis], we identified [new requirement or change]. This was not part of the original scope, and I want to flag it before proceeding.

**What changed:**
[1-2 sentences describing the change]

**Impact:**
- Timeline: [X days/weeks added]
- Budget: [$X additional] or [Absorbed within current scope]
- Trade-off: [If we do this, we deprioritize X]

**Recommendation:**
[Your suggested path forward]

Can we discuss this [on our next call / this week]? I want to make sure we're aligned before moving forward.

Best,
[Your Name]
```

### Invoice Follow-Up

[Your Name]
```

## Follow-Up Cadence

### Cold Outreach

| Touch | Timing | Approach |
|-------|--------|----------|
| 1 | Day 0 | Initial email — value proposition |
| 2 | Day 3 | Reply to original — add new value (case study, data point) |
| 3 | Day 7 | New angle — different pain point or use case |
| 4 | Day 14 | Breakup email — "Is this not a priority right now?" |

**Rules:**
- Always reply to the original thread (builds context)
- Each follow-up adds NEW value, not just "checking in"
- After 4 touches with no response, stop and re-approach in 60-90 days
- Never guilt-trip ("I've emailed you 3 times...")

### Client Communication

| Scenario | Follow-Up Timing |
|----------|-----------------|
| Deliverable sent, no response | 3 business days |
| Proposal sent | 5 business days |
| Invoice overdue | 3 days after due date, then weekly |
| Meeting scheduled, no confirmation | Day before |
| Scope change proposed | 2 business days |

## Closing Phrases by Context

| Context | Options |
|---------|---------|
| **Formal** | Sincerely, Best regards, Respectfully |
| **Professional** | Best, Thanks, Kind regards, Regards |
| **Casual** | Cheers, Thanks!, Talk soon, -[Name] |
| **After a favor** | Appreciate it, Thanks for your help |
| **Urgent** | Thanks in advance, Appreciate the quick turnaround |

## Composition Checklist

- [ ] Subject line is specific (not "Update" or "Hello")
- [ ] Opening sentence establishes context
- [ ] Body is under 150 words (or justified if longer)
- [ ] One clear CTA
- [ ] Tone matches the relationship
- [ ] Proofread for typos and tone
- [ ] Correct recipient (check before sending)
- [ ] Attachments actually attached (if mentioned)
- [ ] No sensitive/confidential info if CC'd broadly
- [ ] Signature block is appropriate
