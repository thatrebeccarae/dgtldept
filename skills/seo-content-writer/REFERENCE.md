# SEO Content Writer — Reference

## SEO Checklist

### On-Page SEO

- [ ] Primary keyword in title tag (front-loaded)
- [ ] Primary keyword in H1 (only one H1 per page)
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in meta description
- [ ] Secondary keywords in H2 headings
- [ ] Keyword density: 1-2% primary, <1% each secondary
- [ ] Title tag: 50-60 characters
- [ ] Meta description: 150-160 characters
- [ ] URL slug contains primary keyword (lowercase, hyphens)
- [ ] At least 2 internal links to related content
- [ ] At least 1 external link to authoritative source
- [ ] Images have descriptive alt text with keywords where natural
- [ ] Header hierarchy is logical (H1 → H2 → H3, no skips)

### Readability

- [ ] Paragraphs: 2-4 sentences max
- [ ] Sentences: 15-25 words average
- [ ] Flesch-Kincaid score matches target audience
- [ ] Active voice >80% of sentences
- [ ] Transition words used between sections
- [ ] Bold/italic used to highlight key phrases (not overused)
- [ ] Bullet lists for 3+ related items

## Platform Specifications

### LinkedIn

| Spec | Value |
|------|-------|
| Post character limit | 3,000 |
| Optimal length | 150-300 words (1,200-1,800 chars) |
| Hook | First 2-3 lines (before "see more") |
| Hashtags | 3-5, mix of broad + niche |
| Image size | 1200x627 (link preview), 1080x1080 (square) |
| Best posting times | Tue-Thu 8-10am, 12pm local |
| Engagement drivers | Personal stories, contrarian takes, data/numbers, questions |

**Structure:**
```
[Hook — provocative statement or number]

[2-3 short paragraphs with the insight]

[Key takeaway or framework]

[CTA — question or invitation]

#hashtag1 #hashtag2 #hashtag3
```

### Twitter/X

| Spec | Value |
|------|-------|
| Tweet limit | 280 characters (premium: 25,000) |
| Thread length | 5-12 tweets optimal |
| Image size | 1600x900 (16:9), 1080x1080 (square) |
| Best posting times | Mon-Fri 9am-12pm, 5-6pm |
| Engagement drivers | Threads, hot takes, useful frameworks, quote tweets |

**Thread structure:**
```
Tweet 1: Hook (standalone value, makes people want more)
Tweet 2-N: One idea per tweet, each can stand alone
Last tweet: Summary + CTA + "Follow for more on [topic]"
```

### Instagram

| Spec | Value |
|------|-------|
| Caption limit | 2,200 characters |
| Optimal caption | 125-150 words |
| Hashtags | 5-15 (mix of sizes) |
| Image sizes | 1080x1080 (feed), 1080x1920 (Stories/Reels) |
| Carousel | Up to 10 slides, 1080x1080 or 1080x1350 |
| Best posting times | Mon, Wed, Fri 11am-1pm |
| Engagement drivers | Carousels, Reels, save-worthy content, questions in captions |

### Email Marketing

| Spec | Value |
|------|-------|
| Subject line | 30-50 characters (mobile-friendly) |
| Preview text | 40-90 characters |
| Body copy | 100-200 words (promotional), 200-500 (newsletter) |
| CTA buttons | 1 primary, max 1 secondary |
| Image width | 600px (standard), 100KB max per image |
| Best send times | Tue-Thu 10am, 2pm local |

**Subject line formulas:**
- Number + benefit: "5 Ways to Cut Ad Spend by 30%"
- Question: "Is Your Email List Costing You Revenue?"
- How-to: "How to Audit Your Shopify Store in 15 Minutes"
- Urgency + value: "Last Day: Free GA4 Audit Template"

## Content Frameworks

### Blog Post: How-To

```
Title: How to [Achieve Result] in [Timeframe/Steps]

H1: [Title with primary keyword]

Intro (100-150 words):
- Hook: Why this matters / what problem it solves
- Promise: What the reader will learn
- Credibility: Brief authority signal

H2: What Is [Topic]? (if needed for informational intent)
- Definition
- Why it matters

H2: Step 1: [Action Verb + Specific Task]
- Explanation
- Example or screenshot
- Pro tip or common mistake

H2: Step 2: [Action Verb + Specific Task]
...

H2: Common Mistakes to Avoid
- Mistake 1 + why it matters
- Mistake 2 + why it matters

H2: FAQ (optional, good for featured snippets)
- Q: [Common question in natural language]
- A: [Concise answer, 2-3 sentences]

Conclusion (50-100 words):
- Recap key steps
- CTA (download, subscribe, try the tool)
```

### Blog Post: Listicle

```
Title: [Number] [Adjective] [Thing] for [Audience/Goal]

H1: [Title]

Intro: Why this list matters, what criteria were used

H2: 1. [Item Name]
- What it is
- Why it made the list
- Key stat or example

H2: 2. [Item Name]
...

H2: How to Choose the Right [Thing]
- Decision framework or comparison table

Conclusion + CTA
```

### Blog Post: Thought Leadership

```
Title: [Contrarian Take or Bold Claim]

H1: [Title]

Intro: Set up the tension — conventional wisdom vs. your position

H2: The Current State of [Topic]
- What most people believe/do
- Data supporting the status quo

H2: Why [Conventional Wisdom] Is Wrong/Incomplete
- Your evidence
- Real-world examples

H2: A Better Approach
- Your framework or recommendation
- How to implement it

H2: What This Means for [Audience]
- Practical implications
- First steps

Conclusion: Restate the bold claim with evidence summary
```
